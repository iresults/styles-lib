## Installation

### composer

```json
{
    "require": {
        "iresults/styles-lib": "*"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://git.iresults.li/git/iresults/styles-lib.git"
        }
    ]
}
```
